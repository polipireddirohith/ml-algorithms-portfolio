## 3. Add to "Featured" Section on Profile

1. Go to your Profile.
2. Find the **Featured** section (or "Add profile section" > "Recommended" > "Add featured").
3. Click the **+** icon and select **Add a link**.
4. Paste your GitHub URL: `https://github.com/polipireddirohith/ml-algorithms-portfolio`
5. Edit the title to: "ML Algorithms Portfolio (10 Projects)"
6. Click Save.
